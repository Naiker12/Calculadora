# **App Name**: NumSolver

## Core Features:

- Bisection Method: Calculate roots of equations using the Bisection method.
- False Position Method: Calculate roots of equations using the False Position method.
- Fixed-Point Iteration Method: Calculate roots of equations using the Fixed-Point Iteration method.
- Gauss-Jordan Elimination: Solve systems of linear equations using Gauss-Jordan elimination.
- Gauss-Seidel Iteration: Solve systems of linear equations using Gauss-Seidel iteration.
- Jacobi Iteration: Solve systems of linear equations using Jacobi iteration.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) for a modern, sophisticated feel, reflecting precision and calculation.
- Background color: Light lavender (#E9DDF7), providing a gentle, uncluttered backdrop that is easy on the eyes during complex calculations.
- Accent color: Muted violet (#8474BE) to draw the user's attention to interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, machined, objective, neutral look.
- Lucide-react icons to represent different numerical methods, providing a visual cue for navigation.
- Implement a sidebar or navbar for navigation between different methods, ensuring ease of access.
- Framer Motion to add subtle transitions and animations, enhancing user experience without distracting from the core functionality.
